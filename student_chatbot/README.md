# 📚 StudyBot — Mini ChatGPT-Style Chatbot for Students

An AI-powered educational chatbot built using **Python**, **FastAPI**, **Wikipedia API**, and **Hugging Face Transformers**.

StudyBot helps students quickly understand educational topics using summarized answers from Wikipedia and NLP-based text summarization.

## ⚙️ Installation Steps

### 1️⃣ Clone Repository

```bash
git clone https://github.com/chussainbee2026-commits/Student_Chatbot_API.git
```

### 2️⃣ Move Into Project Folder

```bash
cd Student_Chatbot_API
```

### 3️⃣ Create Virtual Environment

```bash
python -m venv venv
```

### 4️⃣ Activate Environment

#### Windows
```bash
venv\Scripts\activate
```

#### Linux/Mac
```bash
source venv/bin/activate
```

### 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 6️⃣ Run Application

```bash
python main.py
```
